# Todo.txt notifyer for i3blocks

This is a small perl scripts parsing through todo.txt tasks and display (A) priority and due tasks.

## ~/.config/i3blocks/config

```
[todo]
command=~/bin/blocks/todo-notify.pl
interval=1800
min_width=XXX
```
